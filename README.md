- Band, mode, power, filter, VFO controls
- FlexKnob USB knob controller support
- DX Cluster with auto-band tracking
- Audio recording with PTT-triggered auto-record
- WaveLog integration
- ATU / TG-XL / Amp Tune controls
- Built-in API server
- CW/voice keyer support
