# HamDeck v2.0 - Ham Radio Control for Yaesu FTDX-101MP

By WA0O

## Installation

1. **Right-click** the zip file → **Properties** → check **Unblock** → OK
2. Extract the zip to a folder (e.g. `C:\HamDeck`)
3. Run **HamDeck.exe**
4. No .NET installation required - everything is included

## First Run

- Click **Settings** to configure your radio COM port and baud rate
- Click **Connect** to connect to your FTDX-101MP
- Config file is saved to `%USERPROFILE%\.hamdeck\config.json`

## Features

- Full CAT control for Yaesu FTDX-101MP
- Band, mode, power, filter, VFO controls
- FlexKnob USB knob controller support
- DX Cluster with auto-band tracking
- Audio recording with PTT-triggered auto-record
- WaveLog integration
- ATU / TG-XL / Amp Tune controls
- Built-in API server
- CW/voice keyer support

## Troubleshooting

If Windows blocks the app:
- Right-click HamDeck.exe → Properties → Unblock
- Or: Windows Security → App & browser control → Smart App Control → Off
